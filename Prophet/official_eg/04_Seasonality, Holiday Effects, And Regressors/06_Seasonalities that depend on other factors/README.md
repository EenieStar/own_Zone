# 05_Seasonalities that depend on other factors
# 取决于其他因素的季节性
在某些情况下，季节性可能取决于其他因素，例如在夏季与一年其余时间不同的每周季节性模式，或者在周末与工作日不同的每日季节性模式。
这些类型的季节性可以用条件季节性来建模。
